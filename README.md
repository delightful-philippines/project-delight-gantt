# Project Delight Gantt

A premium, interactive Gantt chart application built with React, Vite, and Supabase.

## Features

- **Interactive Timeline**: Smooth scrolling and zooming.
- **Task Management**: Create, edit, and organize tasks with ease.
- **Resource Allocation**: Manage team members and their assignments.
- **Real-time Sync**: Powered by Supabase for seamless collaboration.
- **Modern UI**: Sleek dark mode and responsive design.

## Tech Stack

- **Frontend**: React, Vite, TailwindCSS
- **Backend**: Node.js, Express
- **Database**: Supabase
- **Authentication**: Azure MSAL / Supabase Auth

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/delightful-philippines/Delightful-Gantt.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   - Copy `.env.example` to `.env`
   - Fill in your Supabase and Azure credentials.

### Development

Run both the frontend and backend:

```bash
npm run dev:all
```
